# Aura 工具使用说明

## 这是什么页面

这个页面是给日常查询用的工具页。

你可以把它理解成 4 个小工具放在同一个页面里：

- 会话查询：按条件筛会话摘要
- 人设查询：查 AI 助手的人设
- 店铺数据查询：查店铺信息
- 统计数据：查看会话统计，还可以导出 Excel

## 先学会这 3 步

第一次用，先记住下面 3 步就够了：

1. 左侧边栏先选环境：`SIT` 或 `STG`
2. 输入商户名称，或者直接输入商户 ID
3. 在左侧工具栏选择你要用的功能，再点对应查询按钮

页面右上角有两个入口：

- `日间 / 夜间模式`：点右上角太阳 / 月亮滑动按钮，切换页面主题
- `会话 UI 图例`：只看会话查询界面怎么读
- `操作指南`：看整页所有功能说明

如果版本刚更新，页面首次进入时会自动弹出一次 `会话 UI 图例`。
日夜间模式会记住你上一次手动选择的结果，刷新页面后继续生效。

如果页面里需要时间范围：

- 开始时间必须早于结束时间
- 最长只能查 7 天

## 每个功能怎么用

### 1. 会话查询

这个功能适合你想先按商户和时间范围把会话筛出来时使用。

如果你第一次看这个界面，先看下面这组分步骤的 UI 图例：

:::session-ui-guide:::

你需要填写：

- 商户
- 开始时间
- 结束时间
- 反馈状态（可不填，从下拉选项里直接选）
- 是否转人工（可不填）
- Email（可不填）

查出来后你能看到：

- 左侧是独立滚动的会话列表，鼠标放在列表区域可直接上下滚动
- 每个会话会展示：
- `sessionId | username`
- `feedbackStatus | isTurnHuman | tagName`
- `lastChatTime | createDt`
- 标签对应关系直接按字段看：
- `feedbackStatus`：状态标签，直接显示状态值，比如 `AI处理中`、`未解决`、`人工介入结束`
- `isTurnHuman`：只有当 `isTurnHuman = true` 时，才会显示橙色标签 `要求转人工`
- `tagName`：灰色分类标签；为空时统一展示成 `Uncategoried`
- 列表底部可以切页
- 右侧点击会话后会调用消息接口，展示类 IM 的聊天详情
- 右侧详情区域本身也是独立滚动，不会把页面无限拉长
- 聊天详情不展示头像，消息正文支持 Markdown 渲染
- `from = CUSTOMER` 的消息会按 `isHuman` 区分标签：`true` 显示 `人工`，`false` 显示 `AI`
- `MERCHANT_EMAIL_FORM`、`CUSTOMER_EMAIL_FORM`、`PRODUCTS` 会按结构化卡片展示，不再只看原始 JSON
- 除 `TEXT`、`COMPLEX` 外，空正文不会再补 `-`
- 会话详情右上角可继续查询当前会话的用户信息

最简单的用法：

1. 选商户
2. 选时间
3. 如果需要，再补充反馈状态、转人工、Email
4. 点“查询会话”
5. 在左侧列表切换页码继续看结果

### 2. 人设查询

这个功能适合你想确认某个商户当前 AI 人设内容时使用。

你只需要：

1. 选商户
2. 点“查询人设”

查出来后，页面会展示这个商户当前的人设配置内容。

### 3. 店铺数据查询

这个功能适合你想看商户关联的店铺基础信息时使用。

你只需要：

1. 选商户
2. 点“查询店铺信息”

查出来后，页面会展示店铺和平台相关信息。

### 4. 统计数据

这个功能适合你想看某个商户最近一段时间的整体表现时使用。

你需要填写：

- 商户
- 开始时间
- 结束时间

查出来后，页面主要会看到两块内容：

- Overview 看板：比如总会话量、AI 解决率、转人工率
- 分类占比看板：展示各分类占多少、各分类有多少

#### 分类占比怎么看

饼图右边每一行都会看到这样的格式：

`百分比 (数量)`

比如：

`35% (128)`

意思就是：

- 这个分类占全部会话的 35%
- 这个分类一共有 128 条

如果只看到百分比，后面的数量是 `-`，说明这条分类没有拿到数量数据。

分类占比看板和导出的 Excel 都会按 `数量` 倒序排列，方便优先看量级最大的分类。

#### 怎么导出 Excel

在“分类占比看板”右上角点 `导出 Excel` 就可以。

导出的表里会带上：

- 商户 ID
- 商户名称
- 查询开始时间
- 查询结束时间
- 分类名称
- 占比
- 数量

如果你刚打开页面还没查数据，先查出统计结果后再导出。

## 常见问题

### 1. 搜不到商户怎么办

可以试两种方式：

- 输入商户名称关键字
- 直接输入商户 ID

### 2. 为什么点查询没反应

一般先检查这几项：

- 有没有选商户
- 开始时间和结束时间有没有填
- 结束时间是不是早于开始时间
- 查询范围是不是超过了 7 天

### 3. 为什么刷新后还是夜间模式 / 日间模式

这是正常行为。

右上角主题滑动按钮会记住你上一次手动切换的模式，所以刷新页面后会继续保持你最近一次的选择。

### 4. 为什么导不出 Excel

先确认：

- 你已经查出了统计数据
- 分类占比区域里已经有内容

如果分类区域是空的，就不会有可导出的内容。

### 5. 为什么右侧只有 `Choose a session`

这是当前版本的预留区域。

本次改造先把会话列表查询、状态色和分页切换稳定下来，右侧详情位后续再补。

## 给第一次使用的人一个建议

如果你不确定该点哪个功能，就按下面理解：

- 想先筛会话，用“会话查询”
- 想看 AI 配置，用“人设查询”
- 想看店铺资料，用“店铺数据查询”
- 想看整体数据和导表，用“统计数据”
