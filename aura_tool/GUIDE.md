# Aura 工具使用说明

## 这是什么页面

这个页面是给日常查询用的工具页。

你可以把它理解成 4 个小工具放在同一个页面里：

- 消息查询：查聊天记录
- 人设查询：查 AI 助手的人设
- 店铺数据查询：查店铺信息
- 统计数据：查看会话统计，还可以导出 Excel

## 先学会这 3 步

第一次用，先记住下面 3 步就够了：

1. 页面顶部先选环境：`SIT` 或 `STG`
2. 输入商户名称，或者直接输入商户 ID
3. 选择你要用的功能，点查询按钮

如果页面里需要时间范围：

- 开始时间必须早于结束时间
- 最长只能查 7 天

## 每个功能怎么用

### 1. 消息查询

这个功能适合你想看某个商户在一段时间内的聊天内容时使用。

你需要填写：

- 商户
- 开始时间
- 结束时间
- Email（可不填）

查出来后你能看到：

- 每个会话的消息列表
- 用户发了什么
- 客服回了什么
- Markdown 格式消息会按标题、列表、引用、代码块、链接等结构展示
- 图片消息
- 用户信息按钮

最简单的用法：

1. 选商户
2. 选时间
3. 点“查询消息”
4. 在结果里展开某个会话查看详情

### 2. 人设查询

这个功能适合你想确认某个商户当前 AI 人设内容时使用。

你只需要：

1. 选商户
2. 点“查询人设”

查出来后，页面会展示这个商户当前的人设配置内容。

### 3. 店铺数据查询

这个功能适合你想看商户关联的店铺基础信息时使用。

你只需要：

1. 选商户
2. 点“查询店铺信息”

查出来后，页面会展示店铺和平台相关信息。

### 4. 统计数据

这个功能适合你想看某个商户最近一段时间的整体表现时使用。

你需要填写：

- 商户
- 开始时间
- 结束时间

查出来后，页面主要会看到两块内容：

- Overview 看板：比如总会话量、AI 解决率、转人工率
- 分类占比看板：展示各分类占多少、各分类有多少

#### 分类占比怎么看

饼图右边每一行都会看到这样的格式：

`百分比 (数量)`

比如：

`35% (128)`

意思就是：

- 这个分类占全部会话的 35%
- 这个分类一共有 128 条

如果只看到百分比，后面的数量是 `-`，说明这条分类没有拿到数量数据。

分类占比看板和导出的 Excel 都会按 `数量` 倒序排列，方便优先看量级最大的分类。

#### 怎么导出 Excel

在“分类占比看板”右上角点 `导出 Excel` 就可以。

导出的表里会带上：

- 商户 ID
- 商户名称
- 查询开始时间
- 查询结束时间
- 分类名称
- 占比
- 数量

如果你刚打开页面还没查数据，先查出统计结果后再导出。

## 常见问题

### 1. 搜不到商户怎么办

可以试两种方式：

- 输入商户名称关键字
- 直接输入商户 ID

### 2. 为什么点查询没反应

一般先检查这几项：

- 有没有选商户
- 开始时间和结束时间有没有填
- 结束时间是不是早于开始时间
- 查询范围是不是超过了 7 天

### 3. 为什么导不出 Excel

先确认：

- 你已经查出了统计数据
- 分类占比区域里已经有内容

如果分类区域是空的，就不会有可导出的内容。

### 4. 为什么有些消息看起来像富文本

消息内容现在支持 Markdown 展示。

如果消息里本身带了这些格式，页面会按结构显示：

- 标题
- 列表
- 引用
- 代码块
- 粗体
- 链接

如果只是普通文本，还是按普通消息展示。

## 给第一次使用的人一个建议

如果你不确定该点哪个功能，就按下面理解：

- 想看聊天内容，用“消息查询”
- 想看 AI 配置，用“人设查询”
- 想看店铺资料，用“店铺数据查询”
- 想看整体数据和导表，用“统计数据”
